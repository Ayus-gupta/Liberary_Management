# Library Management System

A focused and reliable system built to help an admin manage library students with simplicity and order. It assigns seat numbers automatically, keeps monthly payment records organized, and preserves historical activityâ€”all without the need for login or registration.

## Features

- Admin-Only System: No login or registration required. Direct access for quick daily use.
- Student Management: Add students easily; seat numbers are allotted automatically.
- Monthly Payment Tracking: See whether each student has paid their monthly fee.
- History Records: Review past payment details and other stored information.
- Straightforward Workflow: Clean structure designed for dependable long-term use.

## Tech Stack

- Frontend: HTML, CSS, JavaScript (or your preferred stack)
- Backend: Flask
- Database: SQLite

## Usage

- Open the dashboard in your browser.
- Add new students; the system assigns seat numbers automatically.
- Track monthly payments and review history when required.
